# Practical Lab Management System

## Overview
The Practical Lab Management System is a Python-based mini project developed to manage laboratory operations efficiently in colleges or institutes. The system helps teachers and students handle lab assignments, attendance management, and lab record maintenance using CSV files.

The project is menu-driven and works through the command line interface.

---

## Features

### User Authentication
- Sign Up functionality
- Login functionality
- Role-based access:
  - Teacher
  - Student

### Lab Management
- Unix Lab
- ADTL Lab
- PG Lab

### Teacher Functionalities
- Assign lab slots
- Mark attendance
- Update attendance records
- Display attendance records

### Student Functionalities
- View attendance records

### Attendance Management
- Store attendance using CSV files
- Mark Present/Absent
- Record performed tasks
- Update previous attendance records

---
